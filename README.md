
## API Reference

#### Get all items

```http
  GET /api/estudiantes
```

#### Get item

```http
  GET /api/estudiantes/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of item to fetch |
